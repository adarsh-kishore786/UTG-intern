Welcome to QOI - Quite Ok Image

The Quite OK Image Format (QOI) is a compression specification for raster images with 24-bit color (RGB) or 32-bit color with an alpha channel (RGBA).
It is designed to achieve lossless compression and was invented by Dominic Szablewski. QOI aims to reduce the file size of images while preserving the original quality.
